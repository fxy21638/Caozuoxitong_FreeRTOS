# RS485 协议字节序说明

## 总则

**所有多字节数值一律使用大端序（Big-Endian，MSB 在前）。**

## 帧结构

```
Byte:  0      1      2      3      4 ... 4+N-1  4+N   4+N+1
     +------+------+------+------+-----------+------+------+
     |  AA  | Addr | Len  | Type |  Data[N]  | CRC  |  55  |
     +------+------+------+------+-----------+------+------+
```

所有单字节字段（帧头、地址、长度、类型、CRC、帧尾）不受字节序影响。

## 各消息类型的多字节字段

### Type 0x01 — 温湿度 (DHT)

| 字段 | 字节数 | 字节序 | 示例 |
|------|--------|--------|------|
| 温度 (Temp) | 2 | **大端** | 30.0°C → raw=300 → `0x01 0x2C` |
| 湿度 (Humi) | 2 | **大端** | 50.0% → raw=500 → `0x01 0xF4` |

```
温度 = (int16_t)((TempH << 8) | TempL) / 10.0
湿度 = (int16_t)((HumiH << 8) | HumiL) / 10.0
```

**完整帧示例：** `AA 01 05 01 01 2C 01 F4 xx 55` → 温度 30.0°C, 湿度 50.0%

| 字节位置 | 值 | 说明 |
|----------|-----|------|
| 0 | AA | 帧头 |
| 1 | 01 | 目标地址（管理端） |
| 2 | 05 | 长度（Type 1 字节 + Data 4 字节） |
| 3 | 01 | 消息类型（温湿度） |
| **4** | **01** | **温度高字节 (TempH)** |
| **5** | **2C** | **温度低字节 (TempL)** |
| **6** | **01** | **湿度高字节 (HumiH)** |
| **7** | **F4** | **湿度低字节 (HumiL)** |
| 8 | xx | CRC-8（CRC 覆盖 Byte 0~7） |
| 9 | 55 | 帧尾 |

解析：`Temp = (0x01 << 8 | 0x2C) = 0x012C = 300 → 300/10 = 30.0°C`

### Type 0x02 — 姿态角 (MPU6050)

| 字段 | 字节数 | 字节序 | 示例 |
|------|--------|--------|------|
| Pitch | 4 (float) | **大端** (IEEE 754 标准网络序) | 15.0f → `0x41 0x70 0x00 0x00` |
| Roll  | 4 (float) | **大端** | -30.0f → `0xC1 0xF0 0x00 0x00` |
| Yaw   | 4 (float) | **大端** | 45.0f → `0x42 0x34 0x00 0x00` |

```
// C 代码解析 (memcpy 直接拷贝，float 内部小端，但帧上是大端)
float pitch, roll, yaw;
memcpy(&pitch, &f->data[0], 4);  // 帧上是大端，需翻转
memcpy(&roll,  &f->data[4], 4);
memcpy(&yaw,   &f->data[8], 4);
```

> **注意：** 当前 C 代码直接用 `memcpy` 读 float，**未做字节序翻转**。如果采端按大端发送，管理端需先翻转再 memcpy，或者采端统一按小端发送。**需要和队友确认。**

**完整帧示例（大端 float）：** 
```
AA 01 0D 02 41 70 00 00 C1 F0 00 00 42 34 00 00 xx 55
```

### Type 0x03 — LED 控制

| 字段 | 字节数 | 字节序 | 说明 |
|------|--------|--------|------|
| LED 状态 | 1 | 无（单字节） | 0=灭, 1=亮 |

## CRC-8 校验

- 多项式：`x^8 + x^2 + x + 1` (0x07)
- 覆盖范围：帧头 (0xAA) 到 CRC 前一字节（不含帧尾）
- 初始值：0x00
- 算法：查表法

```
CRC = CRC8_Compute(&frame[0], 4 + Len)
// frame[0]=AA, frame[1]=Addr, frame[2]=Len, frame[3]=Type, frame[4..]=Data
```

## 速查表

| 类型 | 帧示例 | 含义 |
|------|--------|------|
| DHT 请求 | `AA 02 01 01 A7 55` | 向采端1请求温湿度 |
| DHT 应答 (30°C/50%) | `AA 01 05 01 01 2C 01 F4 27 55` | 温度 30.0°C, 湿度 50.0% |
| MPU 请求 | `AA 03 01 02 C5 55` | 向采端2请求姿态角 |
| MPU 应答 (15/-30/45) | `AA 01 0D 02 41 70 00 00 C1 F0 00 00 42 34 00 00 7E 55` | Pitch 15°, Roll -30°, Yaw 45° |
| LED 控制 (亮) | `AA 01 02 03 01 D6 55` | LED 点亮 |
| LED 控制 (灭) | `AA 01 02 03 00 3E 55` | LED 熄灭 |
